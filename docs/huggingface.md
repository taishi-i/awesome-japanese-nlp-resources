## Models
### text-generation
 * [open-calm-3b](https://huggingface.co/cyberagent/open-calm-3b) - 📥 229k / ⭐ 21 / OpenCALM is a suite of decoder‑only Japanese transformer language models (160 M–6.8 B params) released by CyberAgent, Inc. under CC‑BY‑SA 4.0, trained on Japanese Wikipedia and Common Crawl, and usable via Hugging Face’s torch‑transformers.
 * [NVIDIA-Nemotron-Nano-9B-v2-Japanese](https://huggingface.co/nvidia/NVIDIA-Nemotron-Nano-9B-v2-Japanese) - 📥 155k / ⭐ 136 / A 9‑billion‑parameter Japanese‑optimized LLM, NVIDIA Nemotron‑Nano‑9B‑v2‑Japanese, trained on data up to September 2024 with a hybrid Mamba‑2/MLP/4‑layer‑attention architecture, fine‑tuned on the Nemotron‑Personas‑Japan tool‑calling dataset, that optionally generates a controllable reasoning trace before producing a final answer and is available for commercial use.
 * [japanese-gpt-neox-small](https://huggingface.co/rinna/japanese-gpt-neox-small) - 📥 147k / ⭐ 15 / A 12‑layer, 768‑hidden Japanese GPT‑NeoX model trained on CC‑100, C4, and Wikipedia, compatible with Huggingface, with an optional toy prefix‑tuning weight that forces each sentence to end with a smiling‑face emoji.
 * [gpt-neox-japanese-2.7b](https://huggingface.co/abeja/gpt-neox-japanese-2.7b) - 📥 94k / ⭐ 59 / A 2.7‑B‑parameter Japanese GPT‑NeoX model trained on Japanese CC‑100 and OSCAR by ABEJA Inc, usable via Hugging Face Transformers pipelines or PyTorch and released under the MIT license.
 * [karasu-1.1B](https://huggingface.co/lightblue/karasu-1.1B) - 📥 51k / ⭐ 7 / Pretrained TinyLlama in Japanese (≈50 k steps) built on ~3 B OSCAR/mC4 tokens, usable via HuggingFace Transformers or VLLM, created by Peter Devine, Sho Higuchi, Yuuki Yamanaka, Atom Sonoda, Shunichi Taniguchi, Tomioka Wataru, and Renju Aoki.
 * [llm-jp-3-150m](https://huggingface.co/llm-jp/llm-jp-3-150m) - 📥 40k / ⭐ 6 / LLM‑jp‑3‑150m—a 150‑M‑parameter Japanese language model from the National Institute of Informatics’ LLM R&D Center—is distributed in Hugging Face Transformers format, requiring torch ≥ 2.3.0, transformers ≥ 4.40.1, accelerate ≥ 0.29.3, flash‑attn ≥ 2.5.8, and was pretrained on Japanese Wikipedia, Common Crawl, WARP/PDF, WARP/HTML, and Kaken data using a unigram byte‑fallback tokenizer.
 * [shisa-gamma-7b-v1](https://huggingface.co/augmxnt/shisa-gamma-7b-v1) - 📥 33k / ⭐ 18 / Fine‑tuned the Japanese Stable LM Base Gamma 7B with Shisa 7B data, achieving strong results on the JA MT‑Bench.
 * [llm-jp-4-8b-thinking](https://huggingface.co/llm-jp/llm-jp-4-8b-thinking) - 📥 16k / ⭐ 38 / Provides the 8 B‑parameter LLM‑jp‑4‑8b‑thinking Japanese language model from NII, trained with pre‑/mid‑training and aligned via SFT/DPO, ready to use with torch‑transformers and detailed cookbook instructions.
 * [llm-jp-4-8b-instruct](https://huggingface.co/llm-jp/llm-jp-4-8b-instruct) - 📥 13k / ⭐ 7 / llm‑jp‑4‑8b‑instruct is a 4.1 B‑parameter Japanese LLM from NII’s LLM‑jp‑4 series, pretrained on large corpora and then fine‑tuned solely with supervised instruction data (no DPO/REINFORCE) and accompanied by cookbook‑style usage guides and a byte‑fallback unigram tokenizer.
 * [Mistral-Nemo-Japanese-Instruct-2408](https://huggingface.co/cyberagent/Mistral-Nemo-Japanese-Instruct-2408) - 📥 11k / ⭐ 48 / A Japanese continually‑pre‑trained Mistral‑Nemo model (Mistral‑Nemo‑Japanese‑Instruct‑2408) built on mistralai/Mistral‑Nemo‑Instruct‑2407, usable via transformers with device mapping and ChatML prompts, released under Apache‑2.0 by Ryosuke Ishigami.
 * [Llama-3-Swallow-8B-Instruct-v0.1](https://huggingface.co/tokyotech-llm/Llama-3-Swallow-8B-Instruct-v0.1) - 📥 11k / ⭐ 21 / Llama3 Swallow is a Japanese‑enhanced Meta Llama 3 family released July 1 2024, offering 8B and 70B variants in Instruct and chat forms fine‑tuned with SFT and Chat Vector on Megatron‑LM, and benchmarked on key Japanese NLP tasks.
 * [Llama-3-70B-japanese-suzume-vector-v0.1](https://huggingface.co/mmnga/Llama-3-70B-japanese-suzume-vector-v0.1) - 📥 8k / ⭐ 4 / Experimental Japanese model created by extracting differences between lightblue/suzume‑llama‑3‑8B‑japanese and Meta‑Llama‑3‑8B‑Instruct using a chat‑vector approach, upsampled and applied to Meta‑Llama‑3‑70B‑Instruct, showing little change and planning future scaling.
 * [Gemma-2-Llama-Swallow-9b-pt-v0.1](https://huggingface.co/tokyotech-llm/Gemma-2-Llama-Swallow-9b-pt-v0.1) - 📥 8k / ⭐ 1 / Japanese‑enhanced, instruction‑tuned Gemma‑2 models built on Llama (2b/9b/27b pre‑train and instruction versions) released on May 19 2025, available via HuggingFace and the Swallow team’s website.

 * [Llama-3-ELYZA-JP-8B](https://huggingface.co/elyza/Llama-3-ELYZA-JP-8B) - 📥 7k / ⭐ 148 / Llama‑3‑ELYZA‑JP‑8B is a Japanese‑enhanced, 8‑billion‑parameter Llama 3 model by ELYZA, fine‑tuned for Japanese on Meta‑Llama‑3‑8B‑Instruct.
